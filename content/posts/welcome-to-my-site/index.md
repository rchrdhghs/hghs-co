## Hello World

Welcome to Hughes Co — a place where I document my homelab adventures, tech experiments, and things I've learned along the way.

## How This Site Works

This site is built with a workflow I'm pretty happy with:

- **Writing** happens in [[Obsidian]]
- **Publishing** is a single hotkey press that pushes content to GitHub via the Enveloppe plugin
- **Building** is handled by Hugo with the Mana theme, triggered automatically by GitHub Actions
- **Hosting** is on GitHub Pages — free, fast, and simple

The entire content pipeline is just markdown files with frontmatter. No database, no CMS backend, no deploy scripts to maintain.

I plan to write about:

- Networking and infrastructure

Stay tuned!
